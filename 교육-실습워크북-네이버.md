# 🟩 오픈클로(openclaw)로 네이버 블로그 자동화 — 실습 워크북

> **대상:** 코딩을 한 번도 해본 적 없는 완전 초보 교육생
> **목표:** 코드를 직접 짜지 않고, **오픈클로에게 한국어로 시켜서** 네이버 블로그 글쓰기를 자동화한다
> **방식:** 회색 박스의 명령을 **그대로 복사 → 오픈클로에 붙여넣기**하며 따라오면 됩니다

> 🔗 **시리즈:** 이 워크북은 **네이버** 편입니다. 같은 시리즈로 **티스토리 편**, **구글 블로거 편**이 있습니다.

---

## ⚠️ 시작 전에 — 네이버는 가장 어렵습니다 (솔직한 안내)

네이버는 세 플랫폼 중 **자동화가 가장 까다롭습니다.** 미리 알고 시작하세요.

| 어려운 이유 | 우리의 해결책 |
|-------------|---------------|
| 🔒 **캡차(보안문자)** 가 자주 뜸 | 사람이 1번만 직접 로그인 → **쿠키(출입증) 재사용** |
| 🪟 글쓰기 칸이 **화면 속 화면(이중 iframe)** | 오픈클로가 단계적으로 프레임을 전환 |
| 🤖 자동화 **탐지가 민감** | 사람처럼 **무작위 시간 간격**으로 천천히 입력 |

> 💡 **마음의 준비:** 한 번에 안 될 수 있습니다. 그래도 괜찮아요. 막히면 오픈클로에게 **"안 돼, 쉬운 말로 설명하고 네가 고쳐줘"** 라고 하면 됩니다.
> 처음이라면 **구글 블로거 편(가장 쉬움)** 이나 **티스토리 편** 을 먼저 해보는 것도 좋습니다.

---

## 🧭 이 워크북을 쓰는 법

각 단계는 이렇게 되어 있어요:

> 📋 **입력할 명령** — 오픈클로에 그대로 붙여넣을 한국어 문장
> ✅ **이렇게 되면 성공** — 무엇이 보이면 잘된 건지
> 🛟 **막히면** — 안 될 때 뭐라고 말하면 되는지

## 📦 준비물 체크리스트

- [ ] 컴퓨터에 **오픈클로(openclaw)** 가 실행됨
- [ ] **네이버 계정** + **네이버 블로그** (없으면 [blog.naver.com](https://blog.naver.com) 에서 무료 개설)
- [ ] 연습용 글 1~2개 (아무 내용 OK)

> ⚠️ **연습은 반드시 "비공개" 글로.** 그리고 **본인 계정으로, 소량만** 합니다.

## 🎬 0차시 — 큰 그림 이해하기 (10분)

네이버 자동화의 순서:

```
[1] 사람이 직접 네이버에 한 번 로그인 (캡차도 직접 통과)
        │
[2] 그 "로그인 상태(쿠키)"를 저장 → 다음부터 재사용
        │
[3] 오픈클로가 그 상태로 글쓰기 화면을 열어 제목·본문 입력 후 발행
        │
[4] 사람처럼 천천히, 소량만 → 안전하게 반복
```

> 📋 **첫 명령 — 개념 확인**
> ```
> 나는 코딩을 전혀 몰라. 네이버 블로그에 글을 자동으로 올리는 걸 너에게 시켜서
> 만들고 싶어. 네이버는 캡차 때문에 어렵다고 들었어. 전체 흐름을 코딩 모르는
> 사람도 알게 4줄로 쉽게 설명해줘.
> ```
> ✅ **성공:** "사람이 한 번 로그인 → 쿠키 저장 → 재사용해서 글쓰기" 같은 설명이 나옴.

---

## 🛠️ 1차시 — 작업 공간 만들기

> 📋 **명령 — 프로젝트 폴더 + 도구 준비**
> ```
> 네이버 블로그 자동 포스팅 연습용 작업 폴더를 만들어줘.
> 자동화에 필요한 프로그램(파이썬, 셀레니움 등)을 네가 알아서 설치하고,
> 다 되면 "준비 끝"이라고 알려줘. 내가 할 일이 있으면 쉽게 알려줘.
> ```
> ✅ **성공:** 설치가 끝나고 "준비 끝" 안내.
> 🛟 **막히면:** `방금 나온 에러를 쉬운 말로 설명하고 네가 직접 고쳐줘.`

---

## ✍️ 2차시 — 올릴 글 준비하기

> 📋 **명령 — 글 보관함 만들기**
> ```
> 네이버에 올릴 글을 보관할 폴더를 만들어줘. 메모장처럼 첫 줄에 제목,
> 그 아래에 본문을 쓰면 그대로 올라가게 규칙을 정하고, 예시 글 1개도 만들어줘.
> ```
> ✅ **성공:** 글 폴더와 예시 글 생성, "첫 줄=제목" 규칙 안내.

> 🧪 **직접 해보기:** 예시 글의 제목·내용을 마음대로 바꿔보세요. 그냥 글쓰기예요!

---

## 🍪 3차시 — 캡차를 피하는 핵심: "쿠키" 저장 ⭐가장 중요⭐

네이버 자동화의 **성패가 갈리는 단계**입니다.

**쿠키 = 로그인된 상태를 기억하는 출입증.**
기계는 캡차를 못 풉니다. 그래서 **사람이 딱 한 번 정상 로그인**해서 출입증을 받아두고, 다음부터 그걸 재사용합니다.

> 📋 **명령 1 — 출입증(쿠키) 저장 기능 만들기**
> ```
> 네이버는 캡차 때문에 자동 로그인이 막혀. 그래서 브라우저를 띄워서 내가 직접
> 네이버에 로그인할 수 있게 해주고(캡차도 내가 직접 통과할게), 로그인이 끝나면
> 그 로그인 상태(쿠키)를 파일로 저장하는 기능을 만들고 지금 실행해줘.
> ```
> ✅ **성공:** 브라우저가 뜨면 **여러분이 직접** 네이버에 로그인(캡차 통과) → 터미널에 Enter → "쿠키 저장 완료".

> 📋 **명령 2 — 다음부터는 출입증으로 자동 로그인**
> ```
> 이제부터는 저장한 쿠키(출입증)로 네이버에 자동 로그인하게 만들어줘.
> 자동화 흔적(webdriver 표시)도 숨겨서 캡차가 덜 뜨게 해줘.
> 쿠키가 만료돼서 안 되면 "쿠키를 다시 저장하세요"라고 친절히 알려줘.
> ```
> ✅ **성공:** 비밀번호 입력 없이 로그인된 상태로 시작됨.
> 🛟 **막히면:** `"쿠키가 만료됐다"고 나와. 출입증 다시 저장하는 걸 실행해줘.`

> 📌 **포인트:** 출입증은 시간이 지나면 만료됩니다. 그때는 명령 1을 다시 한 번만 하면 끝!

---

## 🚀 4차시 — 글쓰기 + 발행 자동화 (SmartEditor 다루기)

네이버 글쓰기 칸은 **화면 속에 또 화면(이중 iframe)** 이 들어있어 까다롭습니다. 오픈클로가 처리합니다.

> 📋 **명령 — 글 올리기 자동화**
> ```
> 로그인된 상태에서 네이버 블로그 글쓰기 화면을 열어서, 글 보관함의 글을 하나
> 가져와 제목과 본문을 입력하고 발행해줘. 네이버 글쓰기 칸은 화면 속 화면
> (이중 iframe, SmartEditor) 구조라 프레임 전환을 단계적으로 처리해줘.
> 사람처럼 보이게 입력 사이에 0.5~2초 무작위 간격도 넣어줘.
> 단, 처음에는 꼭 "비공개"로 올려서 연습부터 하자. 실행해서 결과 보여줘.
> ```
> ✅ **성공:** 글쓰기 화면이 자동으로 채워지고 발행됨 → 블로그에 **비공개 글** 생성.
> 🛟 **막히면:**
> - **"작성 중인 글 복구 팝업이 떠요"** → `글쓰기 들어갈 때 "작성 중인 글 복구" 팝업이 떠. 자동으로 닫게 해줘.`
> - **"본문이 입력이 안 돼요"** → `제목은 되는데 본문이 안 돼. 네이버 SmartEditor가 이중 iframe이래. 프레임 전환을 다시 점검해서 본문이 들어가게 고쳐줘.`
> - **"발행 버튼을 못 눌러요"** → `발행 버튼이나 카테고리 설정 창을 못 찾아. 지금 화면을 다시 확인해서 올바른 위치를 찾아 고쳐줘.`

> 🎉 **여기까지 오면 네이버 자동화 완성!** 글 파일만 바꾸면 계속 올릴 수 있어요.

## ⏰ 5차시 — 안전하게 운영하기 (네이버는 특히 중요)

> 📋 **명령 — 안전장치 + 정기 실행**
> ```
> 네이버는 자동화에 민감하니까, 글 사이에 충분한 시간 간격을 두고 하루 1~2개만
> 올리도록 제한해줘. 그리고 매일 정해진 시간에 자동 실행되게 예약하고,
> 성공/실패 기록(로그)도 남겨줘.
> ```
> ✅ **성공:** "하루 개수 제한 + 정기 실행 예약 완료" 안내 + 기록 파일 생성.

> ⚠️ **경고:** 단기간 대량 발행은 **계정 정지·저품질 판정** 위험이 큽니다. 천천히, 소량만!

## 🆘 부록 A — 막혔을 때 만능 주문 모음

| 상황 | 오픈클로에게 이렇게 말하세요 |
|------|------------------------------|
| 빨간 에러가 떴다 | `방금 나온 에러를 쉬운 말로 설명하고, 네가 직접 고쳐줘.` |
| 캡차가 계속 뜬다 | `캡차가 계속 떠. 내가 직접 로그인해서 쿠키를 저장하는 방식으로 가자.` |
| 본문이 안 들어간다 | `네이버 SmartEditor는 이중 iframe이래. 프레임 전환을 점검해서 본문이 들어가게 해줘.` |
| 복구 팝업이 뜬다 | `글쓰기 진입 시 "작성 중인 글 복구" 팝업을 자동으로 닫게 해줘.` |
| 로그인이 자꾸 풀린다 | `쿠키가 만료된 것 같아. 다시 저장하는 걸 실행해줘.` |
| 가끔 됐다 안 됐다 한다 | `로딩을 충분히 기다리도록(명시적 대기) 고쳐줘.` |
| 뭐가 뭔지 모르겠다 | `지금 상황을 코딩 모르는 사람도 알게 1~2줄로 정리해줘.` |

> 💡 **꿀팁:** "쉬운 말로", "내가 코딩 몰라", "네가 직접 해줘" 를 붙이면 오픈클로가 훨씬 친절해집니다.

## 🧾 부록 B — 용어 사전 (1줄 설명)

| 용어 | 쉬운 뜻 |
|------|---------|
| 오픈클로 | 한국어로 시키면 코딩을 대신 해주는 AI 비서 |
| 셀레니움(Selenium) | 사람 대신 브라우저를 클릭·입력해주는 도구 |
| 쿠키(Cookie) | 로그인 상태를 기억하는 출입증 |
| 캡차(CAPTCHA) | "사람 맞아?" 확인하는 보안문자(기계는 못 풂) |
| iframe | 화면 속에 들어있는 또 다른 화면 |
| SmartEditor | 네이버 블로그의 글쓰기 편집기 (이중 iframe 구조) |

## ⚖️ 부록 C — 꼭 지킬 규칙 (윤리 & 안전)

1. **연습은 비공개 글로.** 공개는 확인 후 마지막에.
2. **본인 계정만, 소량만(하루 1~2개).** 네이버는 특히 민감합니다.
3. **남의 글 복제·대량 스팸 금지.** 저작권·저품질 문제가 생깁니다.
4. **비밀번호·쿠키 파일은 절대 공유 금지.** (출입증을 주는 것과 같음)
5. 자동화는 도구일 뿐, 운영 책임은 본인에게 있습니다.
